# Weather App Information

## Purpose
The purpose of this app is to provide the user with current weather information from their's or any other desired location for the next week in both Celsius and Fahrenheit.

## Functions
**Location Input** - The user is able to input any zip code or city into the provided text box and see the weather info for that location <br>
**Current Weather Info** - The app uses data from WeatherAPI.com for up-to-date forecasts <br>
**8-Day Forecast Layout** - The webpage is formatted to showcase the forecasts of the current day as well as the upcoming week <br>
**Unit Change Button** - The user is able to change the degrees from Celsius to Fahrenheit and vice versa with a provided button <br>

## AI Use
The code for the weather app was built using Grok generative AI. The AI wrote the index.html and app.py files from multiple provied prompts to change the forecast layout and add additional functions for ease of use.
